# ProyectoDungeonTools

Spectrol Tools es una aplicación web orientada a jugadores y Dungeon Masters de **Dungeons & Dragons 5e**, que reúne distintas herramientas digitales para facilitar la gestión de partidas, personajes y recursos de juego.

Este proyecto fue desarrollado como **proyecto académico**, aplicando conocimientos de frontend, backend y bases de datos.

---

## Funcionalidades

Actualmente la aplicación incluye:

- **Dice Tray**
  - Tirador de dados 3D en tiempo real usando Three.js y Cannon.js.
- **Initiative Tracker**
  - Control de turnos, HP y estados durante el combate.
- **Spells & Monsters Compendium**
  - Explorador de conjuros y criaturas con filtros y vista detallada.
- **Sistema de usuarios**
  - Registro e inicio de sesión.
- **Gestión de personajes (en desarrollo)**
  - Creación, edición y eliminación de personajes vinculados a un usuario.

## Tecnologías utilizadas

### Frontend
- HTML5
- CSS3 (diseño personalizado, responsive)
- JavaScript (vanilla)
- Three.js
- Cannon.js

### Backend
- Node.js
- Express.js

### Base de datos
- SQLite

## Créditos de herramientas externas

Este proyecto utiliza recursos y APIs de terceros:

- **D&D 5e API**  
  API pública para datos de criaturas, conjuros y reglas de Dungeons & Dragons 5ª edición.  
  Sitio: https://www.dnd5eapi.co

- **Dice 3D Library**  
  Biblioteca para simulación de dados en 3D utilizada en la herramienta *Dice Tray*.  
  Repositorio: https://github.com/sarahRosannaBusch/dice
